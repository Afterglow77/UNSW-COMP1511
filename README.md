## 📖 Course Overview
COMP1511 introduces the fundamental concepts of computer programming and software design. The course focuses on the **C programming language**, emphasizing control flow, arrays, functions, pointers, manual memory management, and linked lists.

**My Goal:** To build a solid foundation in low-level programming before advancing to object-oriented design and systems programming.

## 📅 Weekly Progress (26T1 Syllabus)

Following the official [COMP1511 26T1](https://cgi.cse.unsw.edu.au/~cs1511/26T1/) week-by-week structure.

> **Note on ordering:** Pointers appear in Week 7, not early on. 26T1 first builds fluency with control flow, structs, arrays, functions, and strings (enough for Assignment 1), then introduces the memory model. Different terms may shuffle topics slightly — this README tracks **26T1**.

### Phase 1: C Foundations
- [x] **Week 1: Getting Started**
  - CSE account, Linux terminal, text editor
  - Compile with `dcc`, run a simple C program
  - Basic I/O (`printf`)
- [x] **Week 2: Input & Branching**
  - `scanf` and arithmetic expressions
  - `if` statements and relational operators
- [x] **Week 3: Loops, Enums & Structs**
  - Complex `if` / nested conditionals
  - `while` loops and nested loops
  - `enum` and `struct`
- [x] **Week 4: Arrays & Functions**
  - 1D arrays and arrays of structs
  - Writing and calling functions
  - Looping over collections

### Phase 2: Arrays, Strings & Assignment 1
- [ ] **Week 5: 2D Arrays & Strings**
  - Functions (continued)
  - 2D arrays
  - Scanning until Ctrl-D
  - Basic string manipulation
  - **Milestone:** Assignment 1 — *CS Chicken*
- [ ] **Week 6: Flexibility Week**
  - No formal lectures / tutorials / labs
  - Extra revision & catch-up (esp. Ass1)

### Phase 3: Pointers, Memory & Linked Lists
- [ ] **Week 7: Pointers**
  - Introduction to pointers
  - Scanning until Ctrl-D
  - Characters, strings, and functions with pointers
- [ ] **Week 8: Dynamic Memory & Linked Lists**
  - Structs + pointers

  - `malloc` / `free`
  - **Milestone:** Assignment 2 — *CS Karting*
- [ ] **Week 9: Advanced Linked Lists**
